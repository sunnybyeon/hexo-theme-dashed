## Installation

```bash
cd path/to/hexo/blog
git clone https://github.com/sunnybyeon/hexo-theme-dashed themes/dashed
cd themes/dashed
npm install
```

And then modify the theme setting in `_config.yml`.

```YAML _config.yml
theme: dashed
```
